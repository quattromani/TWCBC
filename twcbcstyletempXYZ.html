Welcome to the Developer Toolkit

This site covers the “how” of creating digital marketing content for for Time Warner Cable Business Class (TWCBC) on the Web. In this section you will find specifics on developing web page components and ensuring good alignment with the prescribed style of TWCBC.

When used properly, this site will support our efforts to follow the “Golden Rule” of code: All code in any part of the code base should look like a single person typed it, no matter how many people contributed.

Toolkit Sections

The toolkit is divided into five different sections.

Primary Elements

This section of the toolkit is made up of the building blocks of a website, established in the context of Time Warner Cable Business Class. Text, lists, buttons, and other basic site elements are included here.

View Primary Elements

Component Library

This section of the site makes use of the primary elements in a myriad of different ways: text and links become breadcrumbs, for instance. They are collected here in these iterations to help ensure consistency.

View Component Library

Helpers

These “helpful” items provide reusable styling for elements and components. Here, you might find a way to consistently center content or quick ways to customize site pages for different devices, all without making new styles or needing assistance from Techno.

View Helpers

Accessibility

No matter our customers’ abilities, we must make sure that everyone can make the most of our site. To help customers who may not be able to see or here, we adopt certain habits when building site pages to help them access our services.

View Accessibility

Best Practices

This section is a great place to start if you are new to working on the TWCBC website or if you are trying to familiarize yourself with the basic principles and approaches to building webpages in the CQ system.

View Best Practices

Questions? Concerns?

This style guide should serve as the arbiter in style disputes: attempts have been made to vet recommendations included here among key stakeholders within TWCBC and guidance here should be considered a “final ruling.”

For more information or to ask questions, refer to the Working with Digital Web Marketing section of this site
