Lists

Unordered and ordered lists are composed virtually the same. The unordered list is used for non-sequential ordering while the ordered list is for specific ordering and are preceeded by numbers by default. I start out by stripping all styling from any lists, including padding and margin. Bulleted lists are then classed as "bulleted", etc. Lists also get classes added to the first and last item in each (.js-first / .js-last), this gets used for removing any border that we don't want to appear at the top or bottom in particular.
